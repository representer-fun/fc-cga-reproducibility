# Writing Checklist

## Before Drafting

- [ ] Read `PAPER_STORY.md`.
- [ ] Read `CLAIMS_TO_EVIDENCE.md`.
- [ ] Decide final title.
- [ ] Decide which tables fit in the main 9-10 pages.
- [ ] Move graph and failure-mode tables to appendix unless space allows.

## Must Include In Main Paper

- [ ] Validity audit.
- [ ] Repaired-baseline comparison.
- [ ] Held-out layering AUPRC.
- [ ] Alert-budget metrics.
- [ ] Bootstrap significance.
- [ ] Few-shot adaptation.
- [ ] New-pair shift.
- [ ] Limitations paragraph.

## Must Not Overclaim

- [ ] No universal SOTA language.
- [ ] No claim that graph neural models improved.
- [ ] No claim that all AML typologies benefit.
- [ ] No claim that repair is useless; repair is useful but insufficient for
      main typology transfer.

## Suggested Main Figures

- [ ] Validity artifact bars.
- [ ] Layering delta CI.
- [ ] Alert-budget bar chart.
- [ ] Few-shot curve.
- [ ] Temporal bucket curve.

## Suggested Appendix

- [ ] Full-data results.
- [ ] Label scarcity.
- [ ] Rho sensitivity.
- [ ] Mechanism analysis.
- [ ] Graph confirmation.
- [ ] Structuring/integration failure modes.
